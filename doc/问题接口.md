## 首页渲染

- 请求地址：`http://127.0.0.1:3000/question/index`

- 请求方式：`get` 

- 请求参数：`Json` 

  | 参数            | 说明   |
  | ------------- | ---- |
  | `token` （请求头） | 令牌   |

- 返回值：`json` 

  | 状态码      | 说明     | 格式                          |
  | -------- | ------ | --------------------------- |
  | `q200`   | 查询成功   | `{"code":"q200","data":[]}` |
  | `q301`   | 该类别无问题 | `{"code":"q301"}`           |
  |          |        |                             |
  | `err602` | 令牌失效   | `{"code":"err602"}`         |
  | `err604` | 令牌无效   | `{"code":"err604"}`         |
  | `err601` | 逻辑错误   | `{"code":"err601"}`         |

  ```json
  "data":[
    {
      "answerer": {
        "id": "回答者ID",
        "name": "回答者昵称",
        "describe": "回答者描述",
        "icon": "头像地址"

      "question": {
        "id": "问题ID",
  	  "type": "类别（专业）",

        "time": "发布时间"
      },
      "answer": {
        "id": "回答ID",
        "link": "回答内容（链接）",
        "time": "回答时间"

      "quantity": {
        "praise": "点赞数",
  ```

## 提问

- 请求地址：`http://127.0.0.1:3000/question/put`

- 请求方式：`post`

- 请求参数：`Json` 

  | 参数            | 说明   |
  | ------------- | ---- |
  | `token` （请求头） | 令牌   |
  | `title`       | 标题   |
  | `profession`  | 专业   |
  | `link`        | 内容   |

- 返回值：`json` 

  | 状态码      | 说明   | 格式                  |
  | -------- | ---- | ------------------- |
  | `q200`   | 提问成功 | `{"code":"q200"`    |
  | `q404`   | 提问失败 | `{"code":"q404"}`   |
  | `err601` | 逻辑错误 | `{"code":"err601"}` |
