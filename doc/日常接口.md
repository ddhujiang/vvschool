## 写日常

- 请求地址：`http://127.0.0.1:3000/everyday/writing`

- 请求方式：`post` 

- 请求参数：`Json` 

  | 参数            | 说明   |
  | ------------- | ---- |
  | `token` （请求头） | 令牌   |
  | `text`        | 日常内容 |

- 返回值：`json` 

  | 状态码      | 说明   | 格式                  |
  | -------- | ---- | ------------------- |
  | `e200`   | 插入成功 | `{"code":"e200"}`   |
  |          |      |                     |
  |          |      |                     |
  | `err602` | 令牌失效 | `{"code":"err602"}` |
  | `err604` | 令牌无效 | `{"code":"err604"}` |
  | `err601` | 逻辑错误 | `{"code":"err601"}` |


## 查日常

- 请求地址：`http://127.0.0.1:3000/everyday/select`

- 请求方式：`post`

- 请求参数：`Json` 

  | 参数   | 说明   |
  | ---- | ---- |
  | 无    |      |

- 返回值：`json` 

  | 状态码    | 说明   | 格式                          |
  | ------ | ---- | --------------------------- |
  | `e200` | 查询成功 | `{"code":"e200","data":{}}` |
  |        |      |                             |

```json
"data": [
  {
	"id": "用户ID",
	"text": "日常内容"
  }
]
```
